## 🖥️projeto

Esse é um projeto Web Responsivo de uma cápsula do tempo para exibir memorias em uma linha do tempo.

## 🚀 tecnologias

Esse projeto foi desenvolvido durante o NLW da Rockeatseat com as seguintes tecnologias:

-HTML
-CSS
-Git e Github
